                        4일차 11교시
                    

                    

                        

                            

                                
Opal 워크플로우 기획 및 특장점

                                

                                    
공유 워크플로우인 "옷/패션 모델용 프롬프트 이미지 에이전트"의 강점은 제미나이의 텍스트 이해 능력과 이미지 생성 모델의 정교함을 다중 에이전트 노드로 매커니즘화한 것에 있습니다.

                                    
    
                                        
  • 구조 분리(Decoupling): `이미지 첨부`와 `변경 디테일`만 입력하면 ➔ 비전 분석 에이전트가 JSON 구조를 도출 ➔ 제미나이 에이전트가 JSON 내 특정 변수만 변경 ➔ 이미지 생성 에이전트가 구조적 일치성을 유지하며 정밀 마스크 편집을 수행.
    • 
                                        
  • 안전한 리믹스: 우측 상단 Share → Allow access to editor view and remix를 활성화해 타인이 내부 프롬프트와 노드 구성을 복제(Remix)할 수 있게 설정.
    • 
                                    

                                

                                

                                    동일 설계로 만들 수 있는 응용 아이디어
                                    
동일한 입출력 방식을 응용해 "스마트 홈오피스 가구/인테리어 마스터 에이전트"를 설계할 수 있습니다. 오피스 원본 이미지와 '체리를 웜 우드에서 무광 아노다이징 메탈로 교체'라는 디테일을 입력하면 룸 레이아웃 JSON을 비교 매칭하여 인테리어 자산을 빠르게 합성합니다.
